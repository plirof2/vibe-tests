URL Parameters

You can use URL parameters to auto-convert:

  • ?input=programs/game.bas - Fetch code from URL
  • &n=MyGame - Set Spectrum filename
  • &a=10 - Set auto-start line
  • &l=on - Enable label mode
  • &i=10 - Set label increment
  • &s=100 - Set label start
  • &3=on - Set output to +3DOS
  • &r=on - Set output to RAW
  • &p=on - Set output to ZX81 (.P)
  • &o=output.tap - Override output filename
  • &v=on - Verbose mode

Example:

?input=programs/game.bas&n=MyGame&a=10&l=on&i=10&s=100&3=on&o=output.tap
