ZX Spectrum BASIC to TAP Converter

How to use: Enter your ZX Spectrum BASIC code in the textarea below, configure the options, then click "Convert & Download" to generate a .tap file.

Input BASIC Code

Example:
10 PRINT "Hello, World!"
20 BORDER 2
30 CLS

Options

Labels Mode: When enabled, you can use @label: syntax to define labels and refer to them without line numbers.

Example:
@start:
10 PRINT "Hello"
20 GOTO start
