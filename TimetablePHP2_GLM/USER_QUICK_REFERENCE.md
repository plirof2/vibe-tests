# Timetable System - Quick Reference

## Login
1. Open timetable044b.php
2. Enter username and password
3. Click "Login"

## Slot Controls
- **Slot 1 / Slot 2 / Slot 3**: Click to switch between different timetable datasets
- **Save 1 / Save 2 / Save 3**: Click to save current timetable to that slot
- **Copy Slot**: Select "From" slot, select "To" slot, then click "Copy"

## Working with Timetable
1. Drag teachers from Teacher Pool to timetable slots
2. Switch between "Show All Classes" and single class views
3. Teachers are color-coded for easy identification

## Important Notes
- Each slot (1, 2, 3) stores a complete, independent timetable
- Autosave saves your work every 5 minutes automatically
- Use different slots to work on multiple timetables
- Copy slot to backup or duplicate your work

## Logout
Click "Logout" link in the User Info section
